Vim学习笔记

May 30, 2018 工具

参考:

  1. http://easwy.com/blog/archives/advanced-vim-skills-catalog/
  2. http://www.atool.org/vim.php

除了简单的移动光标、编辑功能以及查找以外，还需要先掌握的：

  • 分割窗口
  • 移动窗口
  • 编辑多个文件

移动以及编辑

  1. hjkl : 左下上右
  2. uiop : 撤销 插入 从下行开始 粘贴
  3. C-R : 重做
  4. 数字加方向 : 向这个方向移动这么多次
  5. C-] : 进入标签，C-T或C-O(O:old) : 回跳
  6. x : 删除一个字符
  7. 以单词为单位(跳跃)移动光标
    • w : 下一个单词的开头；W : 下一个非词块的开头
    • b : 上一个单词的开头；B ：上一个非词块的开头
    • e : 下一个单词的结尾；E ：下一个非词块的结尾
    • ge ：上一个单词的结尾；gE：上一个非词块的结尾
    • 以上的指令在使用前加上数字x代表移动x次
  8. 移动到行首或行尾
    • <Home> ： ^(把光标移动到该行的第一个非空字符)；0(移动到该行的第一个字符)
    • <End> ： $\$$(美元符)
  9. 移动到指定的字符
    • f + character ： 从当前位置开始向后寻找本行的字符character并定位到上面
    • F + character ： 从当前位置开始向前寻找本行的字符character并定位到上面
    • t + character ： 从当前位置开始向后寻找本行的字符character并定位到它的前一个字符
    • T + character ： 从当前位置开始向前寻找本行的字符character并定位到它的后一个字符
    • 以上的命令在前面加上数字x代表寻找第x个符合该要求的字符，默认为1
  10. ; ： 重复执行上一条执行的命令；, ： 反向重复
  11. % ： 将光标移动到离它最近的下一个括号的右括号，再使用一次%会移动到左括号上
  12. gg : 回到顶部，G ： 到达最后一行，x + G ： 移动到第x行
  13. - H ： 到达当前缓冲区的首行
    • M ： 到达当前缓冲区的中间行
    • L ： 到达当前缓冲区的尾行
    • x + % ： 到达文件的x%行
  14. C - G ： 获取当前光标所在文件行数的百分数，以及文件名等信息
  15. :set number/nonumber ： 开关行号
  16. :set ruler开关窗口右下角的光标
  17. - C - U ： 相当于PageUp
    • C - D ： 相当于PageDown
  18. 使用两个单引号可以跳回到上一个跳跃的位置
  19. 设置跳跃点(用户可设置的跳跃点最多26个，a-z) ：m + x,在当前位置是指一个不可见的跳跃点x，使用`x可以跳跃到跳跃点x处
  20. 特殊跳跃点：
    • ' 跳转前的光标位置
    • " 最后编辑的光标位置
    • [ 最后修改的开始位置
    • ] 最后修改的结束位置

查找

  1. / + regex ： 使用正则表达式从前往后进行查找符合要求的内容(行内的内容)
  2. ? + regex ： 使用正则表达式从后往前进行查找符合要求的内容(行内的内容)
  3. 在某一格单词上面使用*/# ： 将该词作为正向/反向查找的目标，在命令前加计数x表示查找光标下单词第x次出现的地方
  4. 查找时，使用n可以跳到下一个符合要求的内容，使用N可以跳到上一个符合要求的内容
  5. :set ignorecase/noignorecase ： 设置成在查找的时候是否忽略大小写
  6. :set hlsearch/nohlsearch ： 开关查找高亮

指令模式

  1. 计数前缀x + 指令c : 该指令c重复x次

补全

  1. (omni*): 在输入模式中按补全 <C-x><C-o>

多窗口编辑(常用)

启动多窗口编辑

  1. vim -o file1 file2：打开一个编辑会话，显示为水平分割成两半的窗口，一个文件占用一个窗口。
  2. vim -O file1 file2：和上面相同，不过是垂直分割两半的窗口。
  3. vim -o5 file1 file2：将分配5个相同的窗口，有3个是闲置的。(-o后面加一个数字n，是预分配窗口的形式打开)

如果画面分割后不足以容纳下所有窗口，那么第一个文件会占用大部分窗口，余下的窗口则载入缓冲区，暂时不能看到，但仍可以取用。

多窗口编辑

  1. :split：新建一个窗口，将当前窗口分割成两半，他们显示的是相同缓冲区的内容（同一个文件）。这样就可以在两个窗口中浏览相同的文件了。
  2. :vsplit：创建全新的垂直分割的窗口，同样是显示和当前窗口同一个文件内容。
  3. :split otherfile：打开另一个文件而不是再打开一个当前文件。 当多个窗口编辑同一个文件时，在任意一个窗口进行编辑（例如删除一行），那么所有相同文件的窗口上都会同时反应效果，因为他们共享同一个文件缓冲区。

分割窗口的选项

  1. :[n] split(vsplit) [++opt] [+cmd] [file] 其中：
    1. n：为vim指定在新窗口中显示的行数，且新窗口的大小刚好容纳该行数，新窗口位于画面顶端。
    2. opt：传递vim选项信息给新的窗口会话（请注意，它的前面必须加上两个加号）。
    3. cmd：传入欲在新窗口中执行的命令（请注意，它的前面必须加上一个加号）。
    4. file：指定在新窗口中编辑的文件。
  2. :sview filename：以只读的方式水平分割打开一个新窗口。
  3. sfind [++opt] [+cmd] [file]：和split的运作方式相似，但在path中寻找filename，如果vim未找到文件则不显示，否则打开。

在窗口间游走

在gvim或vim中，在窗口中移动其实非常简单，因为gvim已默认支持鼠标点击来换编辑窗口，而vim中，则可以打开mouse选项为命令、输入、导航都激活鼠标的使用：

:set mouse=a
  1. 按住<C-w>，然后加上hjkl，分别表示像左下上右移动窗口。
  2. <C-w> w：这个命令会在所有窗口中循环移动。
  3. <C-w> t：移动到最左上角的窗口。
  4. <C-w> b：移动到最右下角的窗口。
  5. <C-w> p：移动到前一个访问的窗口。

移动窗口

vim中有两种移动窗口方式，一种只是简单地在屏幕上切换窗口，尺寸维持不变；另一种则是改变窗口的实际布局，还会调整尺寸，以填充它移动的位置。

移动窗口本身（轮换或交换）

  1. <c-w> r：向右或向下方交换窗口。
  2. <c-w> R：向左或向上方交换窗口。
  3. <c-w> x：交换同列或同行的窗口的位置。vim默认交换当前窗口的与它下一个窗口的位置，如果下方没有窗口，则试着与上一个窗口交换位置。亦可在此命令前加上数量，与制定的窗口交换位置。

移动窗口并改变其布局

注：下面的第三个字母都是大写， 1. <c-w> K：移动当前窗口至屏幕顶端，并占用全部宽度。 2. <c-w> J：移动当前窗口至屏幕底端，并占用全部宽度。 3. <c-w> H：移动当前窗口至屏幕左端，并占用全部宽度。 4. <c-w> L：移动当前窗口至屏幕右端，并占用全部宽度。 5. <c-w> T：移动窗口至屏新的现有分页。

调整窗口尺寸

改变当前窗口的尺寸，同时当然也会影响到其他窗口。在gvim和vim中，可以用鼠标点击窗口的顶部白色条直接调整窗口尺寸。

  1. <c-w> =：让所有窗口调整至相同尺寸（平均划分）。
  2. <c-w> -：将当前窗口的高度减少一行，也可在ex命令中，:resize -4明确指定减少的尺寸。
  3. <c-w> +：将当前窗口的高度增加一行。同样在ex命令中，:resize +n 明确指定增加尺寸。
  4. <c-w> <：将当前窗口的宽度减少。
  5. <c-w> >：将当前窗口的宽度增加。
  6. <c-w> |：将当前窗口的宽度调到最大，同样在ex命令中，:vertical resize n 明确指定增加尺寸。

分页编辑

除了使多窗口编辑外，vim还允许创建多个分页（tab），每个分页各有独自的行为。

  1. tabnew filename：打开新分页并编辑新文件（可选的）。如果未指定文件，则只打开新分页，并附上空的缓冲区。
  2. tabclose：关闭当前分页。
  3. tabonly：关闭其他所有的分页。如果其他分页中有修改过的文件，则不会移除该分页，除非设置了autowrite选项。此时，所有修改过的文件都在分页关闭前写入磁盘。

游走分页

可以直接点鼠标来移动到另一个分页，也可以用命令 1. <c-PageDown>：移动到下一个分页 2. <c-PageUp>：移动到上一个分页

关闭与离开窗口

有4种关闭窗口的方式，分别是：离开（quit）、关闭（close）、隐藏（hide）、关闭其他窗口 1. <c-W> q：离开当前窗口。 2. <c-W> c：关闭当前的窗口。 3. <c-W> o：关闭当前窗口以外的所有窗口。
